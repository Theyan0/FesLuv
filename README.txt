INSPIRATION:
Playing Secret Santa with friends brought us so much joy that we wanted to scale this experience 
to create a platform to connect entire communities.

=================================================
WHAT IT DOES:
- Play Secret Santa with strangers.
- Volunteer for local events and activities.
- Compete on leaderboards to foster engagement and festive spirit.

HOW WE BUILT IT:
1. DESIGN:
   - User-friendly interface designed using Figma.
   
2. DEVELOPMENT:
   - HTML and CSS for the frontend.
   - Python Flask for the backend.

=================================================
HOW TO RUN THE WEBSITE:

PREREQUISITES:
Ensure Python is installed on your system.
Download Python here: https://www.python.org/downloads/

STEPS TO RUN:
1. Clone the repository to your local machine:
       git clone https://github.com/Theyan0/theJOYVERSE
       cd theJOYVERSE

2. Install the required libraries:
       pip install -r requirements.txt

   This ensures all the necessary libraries are installed in your virtual environment.

3. Run the application:
       python app.py

4. Open your browser and visit:
       http://127.0.0.1:5000

   Have fun exploring The JoyVerse!
